bool vector_pop_at ( vector_t * vec,
size_t index )

Removes an element from the vector at a specified index.

This function removes the element located at the given index within the vector. All elements following the removed index are shifted one position towards the beginning of the vector.

Parameters
[in]vecPointer to the initialized vector_t structure.
[in]indexIndex of the element to be removed.
Returns
Returns true if the operation was successful, false if the vector is NULL, uninitialized, empty, or if the index is out of bounds.
